## 📌 Sobre o Repositório

Este repositório foi criado para armazenar os meus resumos, exercícios e desafios propostos no **Módulo 1** do curso de **HTML5 e CSS3** do [Curso em Vídeo](https://www.cursoemvideo.com/), ministrado pelo professor Gustavo Guanabara.

Neste primeiro módulo, o foco principal é a construção de uma base sólida sobre a Internet, preparação do ambiente de desenvolvimento e os primeiros passos na marcação (HTML) e estilização básica (CSS).

---

## 📚 O que foi aprendido?

- [x] **Conceitos Iniciais:** Como a internet funciona, o que são protocolos, clientes, servidores, domínios e hospedagem. Diferença entre Front-end e Back-end.
- [x] **Configuração do Ambiente:** Instalação e configuração do **Visual Studio Code** (VS Code) e extensões fundamentais para a produtividade.
- [x] **Estrutura Básica HTML5:** Estrutura raiz de um documento HTML (`<html>`, `<head>`, `<body>`), metadados e configuração de idioma.
- [x] **Semântica e Formatação de Texto:** Títulos (`h1` a `h6`), parágrafos (`p`), quebras de linha (`br`), formatação de texto (negrito, itálico, marcações, citações, etc).
- [x] **Símbolos e Emojis:** Inserção de entidades e emojis no código.
- [x] **Listas:** Listas ordenadas (`ol`), não ordenadas (`ul`) e listas de definição (`dl`).
- [x] **Links e Navegação:** Ligações internas e externas (`a`), download de arquivos e navegação entre as páginas do próprio site (âncoras).
- [x] **Multimídia:**
  - Inserção de Imagens (`img`) estáticas e dinâmicas (`picture` / `source`).
  - Adição de Favicon.
  - Inclusão de Áudios (`audio`).
  - Inclusão de Vídeos (`video`) locais e externos (ex: YouTube, Vimeo).
- [] **Introdução ao CSS3:** A diferença entre conteúdo (HTML) e design (CSS). Utilização de estilos *Inline*, *Internos* (na tag `<style>`) e *Externos* (arquivo `.css`).
